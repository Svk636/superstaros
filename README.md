## 🛠 GitHub Pages Deployment

### Method 1: GitHub Actions (Recommended — Automatic)

1. Create a new GitHub repository named `superstar-os`
2. Upload all files from this folder to the repository root
3. Go to **Settings → Pages → Source → GitHub Actions**
4. Push to `main` branch — it deploys automatically

### Method 2: GitHub Pages from Branch (Manual)

1. Create repository and push all files
2. Go to **Settings → Pages → Source → Deploy from branch**
3. Select `main` branch, `/ (root)` folder
4. Click Save — live in ~2 minutes

### Files Required
```
superstar-os/
├── index.html          ← Main app (539KB, self-contained)
├── manifest.json       ← PWA manifest
├── sw.js               ← Service worker (offline support)
├── .nojekyll           ← Prevents Jekyll processing
├── icons/
│   ├── icon-192.png    ← Android icon
│   ├── icon-512.png    ← Splash screen icon
│   └── icon-180.png    ← iOS icon
└── .github/
    └── workflows/
        └── deploy.yml  ← Auto-deploy workflow
```

## 📲 Installing on Your Phone

### iOS (Safari)
1. Open your GitHub Pages URL in Safari
2. Tap **Share** → **Add to Home Screen**
3. Name it "Superstar OS" → **Add**

### Android (Chrome)
1. Open your GitHub Pages URL in Chrome
2. Tap the **three dots menu** → **Add to Home Screen**
3. Or wait for the install prompt banner

### Desktop (Chrome/Edge)
1. Open the URL
2. Click the **install icon** in the address bar
3. Click **Install**
